## Introduction

The purpose of this assignment is to learn the basics of unit testing.

This assignment has a video walkthrough to accompany the written instructions. Some checkpoints will require you to complete work on your own.

Be sure to match any provided video output exactly, including:

- filenames
- code
- comments
- commit messages
- screenshots

## Learning Goals

- Execute a test suite using the `dotnet test` command.
- Understand the "Arrange-Act-Assert" pattern.
- Utilize the "Red-Green-Refactor" cycle to ensure that tests work as expected.
- Fix a failing test.
